使用Keyboard maestro快捷获得文件的Markdown格式的链接

2021-07-03 1 min read
  1. 安装Keyboard maestro
  2. 下载script
    Copy as Markdown Link - Macro Library - Keyboard Maestro Discourse
  3. 双击导入下载好的script
  4. 在Keyboard maestro中选择“MD Link tools”，右键选择 “Enable Macro Group"
  5. 修改“Copy as Markdown link“的快捷键为”⌃⌥⌘M"，原来默认是”⌥⌘M"，和Mac最小化窗口快捷键有冲突
  6. 在Devonthink等软件中，要获得文件的Markdown格式的链接就按上述快捷键获得
