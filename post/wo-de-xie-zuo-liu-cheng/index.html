我的写作流程

 2020 年 05 月 5日

文献管理：Zotero

写作工具：Scrivener (使用Markdown格式写作)

格式转换工具：Pandoc、DocDown

参考文献：My workflow for transforming academic Markdown into beautiful Word documents

​ Introducing DocDown, the easy way to turn academic Markdown into Word documents

基本流程

第一部分：在文本编辑软件中插入引用。

  1. 下载和安装Zotero。Zotero的使用方法，另外撰文归纳。

  2. 安装Zotero的插件 Zotero Better BibTeX，用于在文本编辑器中插入引用。

    在插入引用时要把输入法切换成英文输入。

  3. 修改Citation keys的格式：打开Zotero的“首选项-Better BibTeX-Citation keys”， 修改格式如下：

    修改后，插入引文后，显示的格式为：liu2020

  4. 在Zotero打开“文件-导出文献库...”，选择如下，不用选择任何选项。导出文件可以任意名称和存放位置。目前文件名是“My Library”，存放地点是“/Users/harvey/OneDrive/papers/Zotero/My Library.bib”。

  5. Now go back to Preferences - Better BibTeX. Go to the Automatic Export tab. You’ll see your Better BibLaTeX library. I changed the Automatic export option here to On change, which means it’ll update my library as soon as I change or add anything to Zotero, but you can keep yours however you want. This finishes the Zotero section of the referencing system.

  6. 下载 zotpick-applescript，使用软件BBEdit打开文件，复制文件内容。这个代码可以是你在任何文本编辑软件（如Scrivener）里插入引文。代码内容（49行）如下：

    # App to keystroke citations from Zotero into Scrivener

# Set a delay between keystrokes for OS Sienna bug
set keyDelay to 0.01

tell application "System Events"
	try
		-- Have to hardcode Scrivener here because it seems to make the script the frontmost app
		set appName to "Scrivener"
	on error errMsg
		display alert "Problem" message "Could not get the name of the frontmost application."
		error number -128
	end try
end tell

set zotRunning to do shell script "/usr/bin/curl 'http://127.0.0.1:23119/better-bibtex/cayw?probe=probe' 2>/dev/null; exit 0"
if zotRunning is "ready" then
	set theReference to do shell script "/usr/bin/curl 'http://127.0.0.1:23119/better-bibtex/cayw?format=pandoc' 2>/dev/null; exit 0"
	try
		repeat until application appName is frontmost
			tell application appName to activate
		end repeat
	on error errMsg
		display alert errMsg
	end try
	tell application "System Events"
		try
			repeat with letter in theReference
				keystroke letter
				delay keyDelay
			end repeat
		on error errMsg
			display alert errMsg
		end try
	end tell
else
	if zotRunning is "" then
		set alertTitle to "Zotero not running"
		set alertMessage to "This script will not work unless Zotero is running. Please launch Zotero and try again."
	else if zotRunning is "No endpoint found" then
		set alertTitle to "Better BibTeX not installed"
		set alertMessage to "This script will not work unless Better BibTeX is installed. Please make sure that Better BibTeX is installed in the running instance of Zotero."
	end if
	display alert alertTitle message alertMessage
	tell application appName
		activate
	end tell
	error number -128
end if

  7. 在Mac应用程序中打开“自动操作”程序，新建“快速操作”（⚙️），At the top, select no input in any application. Then drag the ‘Run AppleScript’ module from the left sidebar into the main window. Replace everything in the text box with the contents of the ‘zotpick-pandoc.applescript’ you copied. Save your new service! 保存为Zotpick。

  8. 修改Zotero的引用快捷键为：⌘-`。

  9. 经过6-8步骤后，在撰写文章需要引用时，按快捷键：⌘-`，调出Zotero的搜索框，若要搜索中文文献切换到“经典视图”，

    另外插入中文引用时，需要把输入法切换成英文。

第二部分 将Markdown格式转换成Word格式

以上步骤解决了引用的问题。下面解决转换问题

  1. 下载安装 Pandoc。

  2. 下载安装最新版的 DocDown。可使用该软件将markdwon格式文件安装需要的格式转换成Word文档。

第三部分 使用Scrivener撰写文章

使用Scrivener撰写文章，在需要的地方使用快捷键 **⌘-`**插入引文。也可以使用Scrivener整理引用的文献内容，具体可以参考这篇文章。文章撰写完成后，编译为“Basic MultiMarkdown”格式，然后使用DocDown软件转换为Word文档。
